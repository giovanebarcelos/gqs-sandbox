Enunciado da Aplicação: Calculadora de Operações Básicas

Desenvolva uma calculadora simples que seja capaz de realizar operações básicas de adição, subtração, multiplicação e divisão. A calculadora deve aceitar dois números como entrada e retornar o resultado da operação solicitada. Além disso, a calculadora deve tratar casos de erro, como divisão por zero.

Desenvolva a aplicação em Java e utilize o JUnit para escrever testes unitários que cubram todas as funcionalidades da calculadora.

Demonstração das Possibilidades de Teste Unitário Utilizando JUnit:

Aqui está a implementação da classe Calculadora e os testes unitários correspondentes:

Classe Calculadora:
public class Calculadora {
    public int somar(int a, int b) {
        return a + b;
    }

    public int subtrair(int a, int b) {
        return a - b;
    }

    public int multiplicar(int a, int b) {
        return a * b;
    }

import java.util.ArrayList;
import java.util.List;

public class TaskList {
    private List<Task> tasks;

    public TaskList() {
        this.tasks = new ArrayList<>();
    }

    public void addTask(Task task) {
        tasks.add(task);
    }

    public void removeTask(Task task) {
        tasks.remove(task);
    }

    public void markTaskAsCompleted(Task task) {
        task.setCompleted(true);
    }

    public List<Task> getAllTasks() {
        return tasks;
    }

    public List<Task> getIncompleteTasks() {
        List<Task> incompleteTasks = new ArrayList<>();
        for (Task task : tasks) {
            if (!task.isCompleted()) {
                incompleteTasks.add(task);
            }
        }
        return incompleteTasks;
    }

    public List<Task> getCompletedTasks() {
        List<Task> completedTasks = new ArrayList<>();
        for (Task task : tasks) {
            if (task.isCompleted()) {
                completedTasks.add(task);
            }
        }
        return completedTasks;
    }
}

Aqui está um exemplo de como as classes e os testes unitários podem ser estruturados:

java

import static org.junit.Assert.assertEquals;
import org.junit.Before;
import org.junit.Test;

public class LibraryTest {

    private Library library;

    @Before
    public void setUp() {
        library = new Library();
        // Inicializar a biblioteca com alguns livros e usuários para os testes
        library.addBook(new Book("ISBN123", "Livro 1"));
        library.addBook(new Book("ISBN456", "Livro 2"));
        library.addUser(new User("Usuário1"));
    }

    @Test
    public void testAddBook() {
        assertEquals(2, library.getBooks().size()); // Verifica se há 2 livros inicialmente
        library.addBook(new Book("ISBN789", "Livro 3"));
        assertEquals(3, library.getBooks().size()); // Verifica se um novo livro foi adicionado
    }

    @Test
    public void testRemoveBook() {
        library.removeBook("ISBN123");
        assertEquals(1, library.getBooks().size()); // Verifica se o livro foi removido
    }

    @Test
    public void testBorrowBook() {
        library.borrowBook("ISBN123", "Usuário1");
        assertEquals(1, library.getBorrowedBooks("Usuário1").size()); // Verifica se o livro foi emprestado para o usuário
    }

    @Test
    public void testReturnBook() {
        library.borrowBook("ISBN123", "Usuário1");
        library.returnBook("ISBN123");
        assertEquals(0, library.getBorrowedBooks("Usuário1").size()); // Verifica se o livro foi devolvido pelo usuário
    }

    @Test
    public void testAddUser() {
        library.addUser(new User("Usuário2"));
        assertEquals(2, library.getUsers().size()); // Verifica se um novo usuário foi adicionado
    }

    @Test(expected = IllegalStateException.class)
    public void testBorrowLimit() {
        // Adiciona livros suficientes para exceder o limite de empréstimo para um usuário
        library.addBook(new Book("ISBN789", "Livro 3"));
        library.borrowBook("ISBN456", "Usuário1");
        library.borrowBook("ISBN789", "Usuário1");
    }

    @Test(expected = IllegalArgumentException.class)
    public void testInvalidBorrow() {
        // Tenta emprestar um livro que não está disponível na biblioteca
        library.borrowBook("ISBN789", "Usuário1");
    }

    @Test(expected = IllegalArgumentException.class)
    public void testInvalidReturn() {
        // Tenta devolver um livro que não foi emprestado pelo usuário
        library.returnBook("ISBN123");
    }
}

Neste exemplo, a classe Library é testada para garantir que todas as funcionalidades funcionem conforme o esperado e que os casos de erro sejam tratados corretamente.

import java.util.ArrayList;
import java.util.List;

public class TaskList {
    private List<Task> tasks;

    public TaskList() {
        this.tasks = new ArrayList<>();
    }

    public void addTask(Task task) {
        tasks.add(task);
    }

    public void removeTask(Task task) {
        tasks.remove(task);
    }

    public void markTaskAsCompleted(Task task) {
        task.setCompleted(true);
    }

    public void markTaskAsIncomplete(Task task) {
        task.setCompleted(false);
    }

    public List<Task> getAllTasks() {
        return new ArrayList<>(tasks);
    }

    public List<Task> getCompletedTasks() {
        List<Task> completedTasks = new ArrayList<>();
        for (Task task : tasks) {
            if (task.isCompleted()) {
                completedTasks.add(task);
            }
        }
        return completedTasks;
    }

    public List<Task> getIncompleteTasks() {
        List<Task> incompleteTasks = new ArrayList<>();
        for (Task task : tasks) {
            if (!task.isCompleted()) {
                incompleteTasks.add(task);
            }
        }
        return incompleteTasks;
    }

    public Task getTaskByTitle(String title) {
        for (Task task : tasks) {
            if (task.getTitle().equals(title)) {
                return task;
            }
        }
        return null;
    }

    public void updateTask(Task task, String title, String description) {
        task.setTitle(title);
        task.setDescription(description);
    }
}
