Testes para serem realizados no SikuliX
1. Abrir o Bloco de Notas e Digitar Texto: 
   Abra o Bloco de Notas e digite "Hello, SikuliX!"

Script:
python

from sikuli import *

# Abrir o Bloco de Notas
App.open("notepad.exe")
wait(1)  # Esperar um segundo para o Bloco de Notas abrir

# Digitar texto
type("Hello, SikuliX!")

3. Abrir o Bloco de Notas, Digitar e Salvar um Arquivo: 
   Abra o Bloco de Notas, digite "Teste de automação" e salve o arquivo como "teste.txt"

Script:
python

from sikuli import *

# Abrir o Bloco de Notas
App.open("notepad.exe")
wait(1)

# Digitar texto
type("Teste de automação")

# Salvar o arquivo
type(Key.CTRL + "s")
wait(1)
type("teste.txt")
type(Key.ENTER)

5. Capturar uma Captura de Tela e Salvar: 
   Capture a tela inteira e salve a captura como "screenshot.png"

Script:
python

from sikuli import *

# Capturar a tela inteira

# Salvar a captura de tela
shfile = "C:\\caminho\\para\\salvar\\screenshot.png"
shfile.save(screen)

8. Realizar Login em uma Página Web: 
   Abra o navegador, vá para a página de login e insira as credenciais.

Script:
python

from sikuli import *

# Abrir o navegador e navegar para a página de login
App.open("msedge.exe")
wait(3)
type("https://www.example.com/login" + Key.ENTER)
wait(3)

# Inserir nome de usuário e senha
type("C:\\caminho\\para\\imagem\\username_field.png", "meu_usuario")
type("C:\\caminho\\para\\imagem\\password_field.png", "minha_senha")
click("C:\\caminho\\para\\imagem\\login_button.png")

10. Abrir o Paint e Desenhar uma Linha: 
    Enunciado: Abra o Paint e desenhe uma linha diagonal.

Script:
python

from sikuli import *

# Abrir o Paint
App.open("mspaint.exe")
wait(3)

# Selecionar a ferramenta de desenho
click("C:\\caminho\\para\\imagem\\paint_draw_tool.png")
